### 📌 核心项目

#### [Nexus Media](https://github.com/linyuan0213/nexus-media) ⭐ 主力项目
> 功能强大的媒体库管理工具，提供自动化追剧、资源下载、文件整理和订阅管理等功能，适合 PT 用户和影视爱好者使用。

- **自动化追剧**：智能订阅与下载调度
- **资源整理**：自动重命名、硬链接、目录规范化
- **TMDB 元数据匹配**：精准识别与海报刮削
- **插件系统**：可扩展的架构设计
- **Docker 一键部署**：开箱即用的容器化方案

#### [Nexus Media Web](https://github.com/linyuan0213/nexus-media-web) ⭐ 配套前端
> Nexus Media 前端项目，基于 Vben Admin v5.7.0 构建，提供现代化的媒体库管理界面。

- **现代化 UI**：基于 Naive UI + Tailwind CSS 的精致界面
- **插件框架**：支持前端插件动态加载
- **响应式设计**：适配桌面与移动端
- **实时状态**：WebSocket 推送任务进度
